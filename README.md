# Commands

To run code (after compiling it), type:

```
$ ./cal [[month] year]
$ ./date [+format]
$ ./head [-n number] [file...]
$ ./ln [-s] target-file link-file
$ ./ls [-al] [file...]
$ ./mkdir [-p] [-m mode] dir...
$ ./env
$ ./tail [-f] [ -c number | -n number] [file]
$ ./true
$ ./false
$ ./wc [-c | -m] [-lw] [file...]
$ ./pwd
$ ./cksum [file...]
```
